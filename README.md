# 5G-Authentication
---
<p>
在 Virtual Box 上建立 Linux 虛擬環境，
ubuntu-18.04.5-desktop-amd64.iso / ubuntu-20.04.2.0-desktop-amd64.iso
使用 Mininet 拓樸，使用 Ryu 控制傳輸，模擬 5G 認證過程。

環境：
Mac(IOS):  Big Sur 11.1
App:  Virtual Box 6.1.16 r140961
Virtual Box:
RAM:  8192 MB
虛擬硬碟大小:  20.0 GB

環境：
Win:  Win10
App:  Virtual Box 6.1.16 r140961
Virtual Box:
RAM:  16384 MB
虛擬硬碟大小:  60.0 GB
</p>
